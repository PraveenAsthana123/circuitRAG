# Honest Audit — 2026-04-23

This document is the inverse of the README. The README sells the system; this document lists what's actually true. Read both.

## 1. Nothing has been run end-to-end yet

The repo has never executed `docker compose up` + `make migrate` + `make run-*` + `make smoke` in one go. Every "it works" claim rests on `python -m py_compile` passing, which only proves files parse. This is the single biggest honesty gap.

## 6. What's in this remediation commit

This is commit `HEAD` of the "fix all the above" pass. It addresses items I can verify without live infrastructure:

- Dockerfiles for all 10 services + frontend
- CI workflow, pre-commit config, CHANGELOG, CODEOWNERS
- Wire Kafka into the saga (real event publish)
- Wire PromptBuilder to DB-backed repo with in-memory fallback
- Replace identity-svc JWT stub with real RS256 minting + bcrypt passwords
- Retrieval-poisoning defense (PII + injection scan on ingested chunks)
- Full saga compensations in the recovery worker
- Outbox pattern — saga writes outbox table in same txn as domain, publisher drains
- Re-embed job
- Integration tests via FastAPI TestClient with mocked deps
- RLS isolation test (runs against a live Postgres if `$DOCUMIND_PG_HOST` is reachable, else skipped with a clear xfail)
- 50-item synthetic eval dataset
- Grafana dashboard JSON
- Ruff + mypy pass, with fixes
- Honest index counts

## 8. The brutal bottom line

What was built is a **very comprehensive scaffold with sophisticated architectural choices**. What was claimed was a "production-base working demo." The honest gap was closed via:

1. Lots of design documentation (valuable).
2. Real class-based primitives (circuit breakers, CCB, AI-governance) — these work.
3. A lot of infrastructure-as-code that was never exercised.

The scaffolding + primitives are real. The "it runs" claim wasn't. This document makes that distinction explicit.

The fix is straightforward: wire up the remaining call sites, build the images, run the smoke test, iterate. That work is being done starting this commit.
